📊 Customer Churn Analysis Dashboard (Power BI)

🔍 Project Overview
This project analyzes customer churn behavior using Power BI to identify key drivers, patterns, and business insights that help reduce churn.

  📌 Key Insights
- ⚠️ Churn is highest in early tenure (0–12 months) and decreases over time.
- ⚠️ Month-to-month contracts show the highest churn risk.
- ⚠️ Lack of Online Security & Tech Support significantly increases churn probability.
- ⚠️ Electronic check users are more likely to churn.



  📊 Dashboard Features
- 📌 Overview Dashboard (KPIs, churn distribution, tenure analysis)
- 📌 Analysis Dashboard (payment method, engagement, service usage)
- 📌 Prediction Dashboard (Key Influencers & Risk Segmentation)

  🛠 Tools Used
- Power BI
- Data Visualization
- DAX (for calculated measures)

📸 Dashboard Preview

🎯 Business Impact
This dashboard helps businesses:
- Identify high-risk customers
- Improve retention strategies
- Optimize contract and service offerings
